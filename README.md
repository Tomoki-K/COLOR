# e-learning 教材

## About
16進数カラーコードについて理解するための教材．

## Getting started
1. clone repository
```
$ cd (your directory)
$ git clone https://github.com/Tomoki-K/e-learning.git
```

2. install packages  
get npm from [here](https://nodejs.org/dist/v6.11.5/node-v6.11.5.pkg) if not yet installed.
```
$ cd e-learning
$ make install
```

3. run!
```
$ make run
```
