## About
16進数カラーコードについて理解するための教材．

## Getting started
1. クローンせぃ
```
$ cd (your directory)
$ git clone https://github.com/Tomoki-K/e-learning.git
```

2. パッケージをインストールせぃ
(npmが入ってなかったら[ここ](https://nodejs.org/dist/v6.11.5/node-v6.11.5.pkg)からインストールせぃ)
```
$ cd e-learning
$ make install
```

3. うぇいそいや
```
$ make run
```
