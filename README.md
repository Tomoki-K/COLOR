## Getting started
1. clone repository
```
$ cd (your directory)
$ git clone https://github.com/Tomoki-K/e-learning.git
```

2. install packages
get npm from [here](https://nodejs.org/dist/v6.11.5/node-v6.11.5.pkg) if not yet installed.
```
$ cd e-learning
$ npm install
```

3. run webpack
```
$ npm run webpack -- --watch
```

4. run app
```
$ open html/index.html
```
